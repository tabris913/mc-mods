# Random Biome Dimensions

バイオーム追加MODのバイオームを活用し、単一バイオームで構成されるディメンションを動的に生成するForge MODです。

## 動作環境

- Minecraft 1.20.1
- Forge 47.x
- Java 17

## 想定バイオームMOD

- Biomes O' Plenty
- Oh The Biomes We've Gone
- Regions Unexplored
- Terralith

## 機能概要

- コマンド `/rbd_generate` で単一バイオームのディメンションを動的に生成（再起動不要）
- スポーンディメンション（バニラプリセット）を初期スポーン先として使用
- ポータルフレームブロック指定によるディメンション間移動
- 中心からの距離に応じたモブ強化システム
- ワールドボーダーによるプレイエリア制限
- 全モブ共通のカスタムルートテーブル
- 日光ダメージ無効化・明るさに依らないモブスポーン
- バイオーム発見進捗システム（MODごとの全バイオーム踏破で達成）

## バージョン履歴

### v1.1.0

- バイオームプールにバニラバイオーム（`minecraft:*`、63種）を追加
- バイオーム除外リストを追加（デフォルト: the_void, エンド系5種）
- バイオーム発見進捗システムを追加
  - RBDディメンション内でバイオームを踏むと「発見」として記録
  - MODごとに設定ファイルのバイオームプールに含まれる全バイオームを発見すると進捗達成
  - 達成時にchallenge風の紫色メッセージを全プレイヤーにブロードキャスト
  - 進捗はプレイヤーごとに個別追跡・永続化
  - 対象MODと進捗名は設定ファイルでカスタム可能

### v1.0.1

- ワールドボーダーが正しく適用されない問題を修正
  - オーバーワールドのボーダー同期リスナーによる上書きを排除
  - ディメンションロード後にティック遅延で確実に適用するよう変更
  - プレイヤーがディメンションに入った際にボーダー情報をクライアントに送信（可視化対応）
- モブの頭上に名前を常時表示
  - バイオームディメンション: 名前 + レベル表記（例: `Zombie Lv.3`）
  - スポーンディメンション: 名前のみ表示
- 強化倍率の計算を累乗方式から線形増加方式に変更
  - Lv.1 を基準（1.0倍）として、レベルごとに設定値分増加
- 経験値ドロップ量を強化段階に応じて増加する機能を追加
  - コンフィグ `xpMultiplier` で倍率を指定可能（デフォルト: 1.2）
- 難易度間隔のデフォルトを500から100に変更
- 攻撃したエンティティのステータス（HP/攻撃力/防御力）をサーバーログに出力するデバッグ機能を追加

### v1.0.0

- 初回リリース
- 単一バイオームディメンションの動的生成
- スポーンディメンション（バニラプリセット）の自動生成
- ポータルによるディメンション間移動
- 中心からの距離に応じたモブ強化（HP・攻撃力・防御力）
- ワールドボーダー自動設定
- カスタムルートテーブルの適用
- 日光ダメージ無効化
- 明るさに依らないモブスポーン
- 帰還時のMODアイテム削除
- コンフィグによる各種パラメータ調整

## コンフィグ

サーバー起動時に `world/serverconfig/randombimedimensions-server.toml` が自動生成されます。

| 項目 | デフォルト | 説明 |
|------|-----------|------|
| worldSize | 1000 | ワールドボーダーの1辺の長さ（ブロック） |
| portalBlocks | 4種 | ポータルフレームブロックのリスト（リスト長=ディメンション数） |
| difficultyInterval | 100 | 難易度が上がる距離間隔（ブロック） |
| healthMultiplier | 1.5 | レベルあたりのHP増加率 |
| attackMultiplier | 1.2 | レベルあたりの攻撃力増加率 |
| defenseMultiplier | 1.1 | レベルあたりの防御力増加率 |
| xpMultiplier | 1.2 | レベルあたりの経験値増加率 |
| biomePool | (各MODのバイオーム) | ディメンション生成に使用するバイオームプール |
| biomeExclude | (the_void, エンド系5種) | 除外するバイオームのリスト |
| advancementMods | (5MOD) | 進捗対象MODと進捗名（`namespace:タイトル`形式） |
